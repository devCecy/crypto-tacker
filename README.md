# Coin Tracker

1~100위 코인의 시세를 보여줍니다.

## 기능

- 1~100위의 코인 리스트 제공
- 코인에 대한 기본 정보 제공
- 10초 마다 코인 가격 갱신
- 일주일간의 코인 시세를 캔들차트로 확인
- 1년 ~ 15분전 코인 시세%변화를 라인차트로 확인
- 다크모드 지원
